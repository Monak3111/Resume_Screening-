# Resume_Screening


# ATS Resume Screening AI

This is a simple web app that checks how well your resume matches a job description.

You upload your resume (PDF), paste the job description, and it gives you a score based on how many skills match.

---

## 🌐 Live Links

- Frontend: [https://resume-screening-frontend-unt1.onrender.com](https://resume-screening-frontend-unt1.onrender.com)  
- Backend: [https://resume-screening1.onrender.com](https://resume-screening1.onrender.com)

---

## 🧠 What it does

- Upload resume (PDF)
- Paste job description
- Get match score
- See matched skills
- See missing skills

---

## ⚙️ Tech used

- React (Vite)
- FastAPI
- PyMuPDF
- Render for deployment

---

## 🚀 How it works

It reads text from your resume, compares it with the job description, and finds common keywords to calculate a match score.

---

## 💡 Note

Works only with PDF resumes.
